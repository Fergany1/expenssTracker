# expenssTracker
Full-featured personal finance tracker. Add income and expense transactions, auto-calculates balance, income and expense totals. Data persists across sessions via localStorage. Includes delete functionality and date stamping per transaction.

# Expense Tracker

A personal finance tracker built with vanilla JavaScript.

## Features

- Add income and expense transactions
- Auto-calculates balance, total income, and total expenses
- Delete any transaction
- Date stamped entries
- Data persists via localStorage

## Tech Stack

HTML, CSS, JavaScript, localStorage

## Link => https://fergany1.github.io/expenssTracker/

## How to Use

1. Enter a description and amount
2. Use a negative number for expenses (e.g. -500)
3. Click Add or press Enter
4. Click Delete to remove a transaction
